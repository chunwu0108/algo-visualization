Description

Sudoku is a logic-based, combinatorial number-placement puzzle. The objective or a solution is to fill each column, each row, and each of the nine 3Ã—3 subgrids contain all of the digits from 1 to 9.
This tool visualizes the backtracking algorithm which solves any solvable Sudoku board. You can start by generating a board with the number of desired filled slots, and click "Solve" to visualize. The speed can be adjusted at anytime.
(Optional) You can also click on each individual slot and enter a number from 1-9 to set. 0 to unset.

Description

Triple DES or Triple Data Encryption Algorithm, is a symmetric-key block cipher, which applies the DES cipher algorithm three times to each data block.
Usage: You can use the Generate button to randomly generate a secret key, share it with your friend via copy+paste OR Show/Hide Key QR to capture a QR code.
After entering a message, click Compute to encrypt it. Or if you have an encrypted message, paste it in the Cipher text block, and click Compute to decrypt.
